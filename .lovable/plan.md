

# إضافة الاسم للمحاضرة والورشة معاً + جعل الحقول إجبارية + إضافة ملاحظات

## الخطوات

### 1. تعديل قاعدة البيانات
- إضافة عمود `notes` (text, nullable) إلى جدول `people` للملاحظات الاختيارية

### 2. تعديل دالة `addPerson` في `AttendancePage.tsx`
- عند إضافة اسم جديد، يتم إدراج **سجلين** في جدول `people`:
  - سجل بفئة `muhadera`
  - سجل بفئة `warasha` (مع `workshop_number`)
- إظهار خيار رقم الورشة دائماً (بغض النظر عن الفئة الحالية) لأن الشخص سيضاف للورشة أيضاً

### 3. جعل الحقول إجبارية
- **الاسم** - إجباري (موجود حالياً)
- **رقم الهاتف** - إجباري
- **العنوان** - إجباري
- **المواليد** - إجباري
- **تاريخ الانضمام** - إجباري
- **التحصيل الدراسي** - إجباري
- **رقم الورشة** - إجباري
- تغيير placeholder من "(اختياري)" إلى نص بدون كلمة اختياري
- إضافة validation قبل الإرسال تمنع الإضافة إذا أي حقل فاضي

### 4. إضافة حقل ملاحظات (اختياري)
- حقل textarea جديد في نموذج الإضافة والتعديل
- يظهر في البروفايل
- يُحفظ في عمود `notes` الجديد

### 5. تعديل البروفايل والتعديل
- عرض الملاحظات في صفحة البروفايل
- إضافة حقل الملاحظات في نموذج التعديل

## التفاصيل التقنية

```sql
ALTER TABLE people ADD COLUMN notes text;
```

```typescript
// إدراج سجلين بنفس الوقت
const baseData = { name, phone, address, birth_date, join_date, education_level, notes };
const records = [
  { ...baseData, category: "muhadera" },
  { ...baseData, category: "warasha", workshop_number }
];
await supabase.from("people").insert(records).select();
```
