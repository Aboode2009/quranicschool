

# وضع الصيانة (Maintenance Mode)

## الفكرة
إضافة وضع صيانة يمنع أي شخص من استخدام التطبيق ويظهر رسالة "تم إيقاف التطبيق من قبل عبد الرحمن". يتم التحكم به من قاعدة البيانات.

## الخطوات

### 1. إنشاء جدول إعدادات في قاعدة البيانات
- جدول `app_settings` يحتوي على صف واحد فيه عمود `maintenance_mode` (boolean)
- سياسة RLS: الكل يقدر يقرأ، بس الأدمن يقدر يعدّل

### 2. تعديل `src/App.tsx`
- عند تحميل التطبيق، يتحقق من قيمة `maintenance_mode` من الجدول
- إذا كانت `true`: يظهر شاشة كاملة فيها رسالة "تم إيقاف التطبيق من قبل عبد الرحمن" مع الشعار
- إذا كانت `false`: يعمل التطبيق بشكل طبيعي

### 3. التفعيل والإلغاء
- لتفعيل وضع الصيانة: تطلب مني وأغير القيمة بقاعدة البيانات
- أو نضيف زر في لوحة الأدمن للتحكم بوضع الصيانة

## التفاصيل التقنية

```sql
CREATE TABLE app_settings (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  maintenance_mode boolean NOT NULL DEFAULT false,
  updated_at timestamptz NOT NULL DEFAULT now()
);
INSERT INTO app_settings (maintenance_mode) VALUES (false);
```
