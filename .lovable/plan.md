

# تصدير بيانات الحضور كملف Excel

## الفكرة
إضافة زر "تصدير Excel" في صفحة الملف الشخصي لكل شخص، بحيث يتم تحميل ملف Excel يحتوي على كل تفاصيل الحضور والغياب.

## التغييرات المطلوبة

### 1. إضافة مكتبة Excel
تثبيت مكتبة `xlsx` (SheetJS) لإنشاء ملفات Excel من المتصفح مباشرة.

### 2. تعديل `src/pages/AttendancePage.tsx`
- إضافة دالة `exportToExcel` تأخذ بيانات الشخص المحدد (`records`) وتنشئ ملف Excel يحتوي على:
  - **ورقة 1 - حضور المحاضرات**: اسم الدرس، التاريخ
  - **ورقة 2 - غياب المحاضرات**: اسم الدرس، التاريخ، العذر
  - **ورقة 3 - حضور الورشات**: اسم الدرس، التاريخ
  - **ورقة 4 - غياب الورشات**: اسم الدرس، التاريخ، العذر
  - **ورقة 5 - ملخص**: إجمالي الحضور والغياب للمحاضرات والورشات
- إضافة زر "تصدير Excel" بأيقونة تحميل في صفحة الملف الشخصي (بجانب زر الحذف أو في الأعلى)
- اسم الملف يكون: `{اسم_الشخص}_تقرير_الحضور.xlsx`

## النتيجة
المستخدم يفتح ملف أي شخص ← يضغط زر "تصدير Excel" ← يتم تحميل ملف Excel كامل بكل التفاصيل.
